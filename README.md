                                  <h3>Introduction</h3>



  <p>Welcome to my TicTacToe game. This project is showing the skills developed durring three weeks of GA's Web Development Immersive Program.</p>

  <p>This projects uses a custom build game engine to play Tic tac toe, we used few technologies that allow the beauty and the abbility to be displayed on your device and to be able to play. we can sign-in with created account and be able to challenge your friends.</p>
  <br><br>





                              <h3>Technologies Used</h3>


      <h4>HTML:</h4>

      <ul>
        <li>Used to build the structure of the game</li>
        <li>Used to named every single tags to be used in a functions of the game</li>
        <li>Used to created the main game</li>
      </ul>
      <br><br>

      <h4>CSS:</h4>

      <ul>
       <li>Used to style the  gameboard</li>
       <li>Used to style the background</li>
       <li>Used to allow the visibility of the page and the game</li>
      </ul>

      <h4>JavaScript:</h4>

      <ul>
        <li>Used to create functios of each box of the game board (cells) to be abble to play</li>
        <li>Used to add class  to represent each player and to give a value of x and o</li>
        <li>Used to play the game after sign in and reset the game</li>
      </ul>

      <h4>JQuery:</h4>

        <ul>
          <li>Used to print out the first players move and the second player moves</li>
          <li>Used event handlers to be call on functions of the game</li>
          <li>Used to display the PlayGame button after sign in and hide it after sign out.</li>
        </ul>

      <h4>Ajax</h4>

        <ul>
          <li>Used to allow all API interration to happen on a single webpage</li>
          <li>Used of curl requests to test ajax funtionality and to keep track of game security</li>
        </ul>
